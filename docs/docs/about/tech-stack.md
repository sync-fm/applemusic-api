# Tech stack

## Runtime

- **TypeScript** 5.6
- **Bun** 1.1+ (Node.js 18+ compatible)

## HTTP & auth

- **axios** for API calls
- **tough-cookie** for session storage
- **TokenStorage** and **AxiosManager** (custom)

## Testing & build

- **Vitest** with coverage
- **tsc** + **tsc-alias**

## Documentation

- **Docusaurus**
- **TypeDoc**
- **Pagefind** + **Canary**

## Tooling

- **Biome**
- **Knip**
- Bun scripts in `package.json`
